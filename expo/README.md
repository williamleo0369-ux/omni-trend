

 OmniTrend (OT)  全球宏观趋势投研监控平台

OmniTrend 是一款专注于全球宏观市场趋势监控的智能投研工具。它通过多维度的实时数据聚合，为理性决策者提供跨市场（A股、港股、美股、外汇）的深度洞察，旨在消除市场噪音，捕捉真正的博弈信号。



项目核心能力

 1. 全球趋势反射镜 (Global Trend Reflector)

 多维监控：内置 18 个核心监控板块，实时演化市场情绪。
 信号系统：自动化提取“活跃信号”、发布“风险预警”，将复杂数据转化为直观的可操作指标。

 2. 全动态实时行情看板

 多市场覆盖：上证指数、深证综指、恒生指数、标普 500、美元指数等核心标的一屏尽览。
 毫秒级更新：基于实时数据流，确保报价与涨跌幅的绝对精准。

 3. 理性投资社区

 内容精选：通过高赞算法（≥20赞）与编辑精选，过滤无效噪音。
 博弈视角：从社会阶层与理性决策角度出发，构建深度投研交流环境。



 技术栈与架构

 前端框架：TypeScript + React Native (Expo) / SwiftUI (基于 Rork 框架开发)
 实时数据：集成 `liveStock` 动态行情接口，实现资产价格的实时双向绑定。
 持续集成：接入 GitHub CI/CD 流程，自动化部署至 Render / Vercel 环境。
 开发工具：利用 Trae 进行 AI 辅助编程，通过 n8n/Gemini 驱动后端自动化逻辑。



 设计理念

 极简主义：采用深色模式（Dark Mode）与高对比度色彩（High Contrast），降低长时间盯盘的视觉疲劳。
 信息层级：严格遵循 UI/UX 逻辑，顶部品牌区（OT）、中部看板区、底部导航区界限清晰，确保用户在极端行情下也能迅速获取核心信息。
